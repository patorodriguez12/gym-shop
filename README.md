## Overview

E-commerce platform built with Next.js, serverless authentication using Supabase
and hybrid rendering, designed for production.

## Tech Stack

- Next.js
- Supabase
- Stripe
- Cloudinary
- Tailwind CSS
- Others that will be added in the future
